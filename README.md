# Shopping Cart
- A practical application of advanced React concepts, focusing on routing, state management, and API integration. The goal is to build a functional, multi-page mock e-commerce site.

# Core Requirements
- Navigation: A persistent Navigation Bar allowing users to move between three distinct pages (Home, Shop, and Cart).
- Data Fetching: Products must be dynamically retrieved from an external source, such as the FakeStore API.
- State Management: The shopping cart must update in real-time, with the item count reflected in the Navbar across all pages.
- Dynamic Functionality: * Shop Page: Product cards with manual input fields and increment/decrement buttons.
- Cart Page: A summary of selected items with the ability to adjust quantities or remove items.

# Built With
- Vite + React
- React Router
- React Libraries: Swiper, react-spinners, lucide-react.
